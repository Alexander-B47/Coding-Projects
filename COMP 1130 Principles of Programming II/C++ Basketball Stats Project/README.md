# Basketball Stats Displayer
This program calculates and displays basketball statistics for a team. It reads player data from a file, processes the data to compute various statistics such as points per game, assists, and rebounds, and then outputs the results in a formatted manner.
## Features
-Reads from a text file containing Basketball players names and their stats separated by a :
-Displays the players' names and stats to the console formatted
-Let's the user choose who's stats that they would like to see

prog6.cpp uses most of what I learned from this class; including: structs, reading from files, formatting, loops, decision structures, user input, etc.
